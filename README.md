# Self-Driving-Car-simulation
Self Driving Car simulation on Udacity Simulator


# dependencies include
- tensorflow
- keras
- socketio
- flask
- pillow
- socketio
- eventlet
- cv2


# link to simulator

# steps
- create an anaconda environment
- activate it
- run the python script
- open the simulator in autonomous mode
